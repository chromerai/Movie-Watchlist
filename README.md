# Movie WatchList App - Solo Project
<hr>

## Scope
Build a movie watchlist app with the following requirements:

## Design
- Follow the provided [Figma design specification](https://www.figma.com/design/hPh5YXocWXhsXJE0nq7qJ7/Movie-Watchlist--Copy-?node-id=0-1&p=f&t=ual37X7BpIDtX09R-0)
- Access the Figma file by clicking the provided image link
- Duplicate the Figma file to work with your own copy

### Search Movies
- Provided a search feature to search and get the list of movies with the similar/same titles
- Present the returned list in a proper manner as defined in the figma file.
- Provided option to add the movies to the watchlist
- If a movie already exists, its shows relevant information like "added to watchlist" inplace of "add-button"

### Watchlist
- Keeps track of the user's added movies
- Added a remove button to remove movies from the list


### Key Requirements Summary
- Follow the design spec
- Render menu options using JavaScript
- Enable adding movies to list via plus button
- Enable removing movies from watchlist via remove button
